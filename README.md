# Dev
1. Clone Project
2. Copy ```env.template``` and rename to ```.env```
3. Execute
```
yarn
  ```
  4. Pull Docker image
  ```
  docker-compose up
  ```
  5. execute 
  ```
  yarn start:dev
  ```
6. visit the web
  ```
  localhost:3000/graphql
  ```
7. execute the _"mutation"_ executeSeed, in order to fulfull the database with information.
